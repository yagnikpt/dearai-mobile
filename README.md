# DearAI Mobile

Dear AI is a conversational AI application built with Expo. It's designed to be a supportive companion, ready to listen and chat.

## Project Setup

### Prerequisites

Ensure you have [Bun](https://bun.sh/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yagnikpt/dearai-mobile.git
   cd dearai-mobile
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Check the `.env.example`

4. Start the development server:
   ```bash
   bun run android
   ```
